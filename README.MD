# Easy-Httpclient
## 简介
本工具主要是简化apache httpclient 使用，更短代码篇幅。

## 使用

maven 依赖
````xml
<dependency>
    <groupId>com.monchstudio.image</groupId>
    <artifactId>easy-httpclient</artifactId>
    <version>1.0.0</version>
</dependency>
````

## 相关依赖

apache httpclient 4.15.3
